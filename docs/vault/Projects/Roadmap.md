# Roadmap

## Phase 2 — More Data Sources (Next)

### SEC Filings
- [x] **Comment Letters (CORRESP)** — SEC→company questions
- [x] **Insider Trading (Form 4)** — CEO/CFO buy/sell patterns
- [ ] **8-K Material Events** — M&A, restatements, exec changes, cybersecurity incidents
- [ ] **10-K/10-Q Section 1A** (Risk Factors) — Extract year-over-year changes. New risks = new problems.
- [ ] **Proxy Statements (DEF 14A)** — Executive compensation trends, board shakeups, shareholder proposals
- [ ] **Schedule 13D/G** — Activist investor accumulations (>5% positions)
- [ ] **Form S-1** — IPO pipeline tracking

### Government Data
- [ ] **FDA Orange Book API** — Real-time drug patent/approval data (replaces static watchlist)
- [ ] **USPTO Patent API** — Auto-expand patent watchlist via CPC classification codes
- [ ] **Bureau of Labor Statistics** — Employment, inflation, wage data (macro signals)
- [ ] **Census Bureau** — Economic census, construction spending (real estate signals)
- [ ] **Federal Reserve** — FOMC minutes, Beige Book, bank stress tests
- [ ] **CFTC COT Reports** — Commitment of Traders (institutional positioning)
- [ ] **EPA Enforcement** — Environmental violations = future litigation costs

### Alternative Data
- [ ] **FCC License Applications** — Spectrum, broadcast, telecom deals
- [ ] **FAA Aircraft Registry** — Corporate jet movements (M&A signals)
- [ ] **NOAA Weather** — Agricultural commodity predictions
- [ ] **SEC Exempt Offerings** — Form D filings = private money moving

## Phase 3 — Intelligence Layer

- [ ] **Portfolio Integration** — Cross-reference alerts with user's holdings
- [ ] **Historical Backtesting** — Do SEC comments actually predict stock drops?
- [ ] **Network Analysis** — Map company→auditor→lawyer relationships
- [ ] **Sentiment Scoring** — NLP on SEC comment tone (aggressive vs. routine)
- [ ] **Correlation Engine** — "When SEC questions X, stock drops Y% in Z days"

## Phase 4 — Product

- [ ] **Web Dashboard** — Real-time feed of flagged filings
- [ ] **API** — Paid API for hedge funds/traders
- [ ] **Mobile App** — Push notifications for HIGH alerts
- [ ] **Weekly Digest** — Free newsletter (SEO play)
- [ ] **Custom Watchlists** — User-defined company/ticker monitoring
